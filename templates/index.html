Welcome To SentiMeant!

Get in-depth sentiment analysis of a user's tweets. SentiMeant goes through thousands of tweets and uses several recurrent neural networks to perform analysis of a twitter user's overall mood, biases/prejudices and more.

Get Started

How It Works

SentiMeant uses python's flask library to run the site. The sentiment analysis relies on several recurrent neural networks. These neural networks are based on code from academic papers and Stanford's CS230 course. This website's code, as well as info and citiations about the methods and algorithms used can be found on the github link below.

View Github

What's Included

Upon querying a user, SentiMeant will download around 3,000 of a user's most recent tweets. From there, analysis on a user's overall sentiment (positive, negative), political leanings (conservative, liberal) and potential biases (racist, sexist, homophobic) will be graphed
